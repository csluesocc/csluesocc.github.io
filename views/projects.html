Software Libre en la Educación Media

Realizado con el objetivo de dar a conocer la ideología y diversos conocimientos del software libre, ya que muchos estudiantes (en este caso, los bachilleres) tienen escasos conocimientos o no tienen conocimiento sobre lo que es, las ventajas que trae al usarlo, entre otros.

Talleres sobre GNU/Linux

Talleres donde se explica que es GNU/Linux, como nace y porque hay que usarlo, tambien se habla un poco sobre las distintas distribuciones (o distros) que existen y cuales son las mas adecuadas para comenzar a migrar al software libre.

Desarrollo de apps para móviles

Desarrollo de aplicaciones para móviles con HTML5, CSS y JavaScript para dispositivos con Firefox OS y Aplicaciones para dispositivos Android.

Diseño Web

Talleres sobre diseño web con HTML5, CSS, JavaScript, Node.js, Ajax, usando algunos frameworks como Django, Twitter Bootstrap, CodeIgniter, implementando librerias JQuery y usando plugins como FancyBox

Electrónica con hardware libre

Proyectos realizados con placas Arduino

Diseño Gráfico

Talleres sobre como realizar diseños vectoriales, vectorización de archivos gráficos, edición de nodos, texto en trazos y mucho más con Inkscape, ademas como retocar fotografias o crear imágenes con efectos en Gimp.
